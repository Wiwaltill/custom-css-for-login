Simple design of the admin login via CSS
 
== Description ==
 
Custom CSS for Login Plugin holds a lot of customization fields to change the layout of the login page of WordPress. You can modify the look and feel of login page completely even the login graphic, colors and many more. You can also include your own CSS.
Custom CSS for Login Plugin would give you and your users a feeling that it is a custom login page and a part of the site layout.
